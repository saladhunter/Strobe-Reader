# Speedreader

A Swift-based macOS application for speed reading.

## Version
1.0

## Getting Started

### Prerequisites
- macOS
- Xcode
- Swift

### Build
```bash
xcodebuild -project Speedreader.xcodeproj -scheme Speedreader -configuration Debug
```

### Run
Open `Speedreader.xcodeproj` in Xcode and build/run.
